## 그래프

탐색기법

1) 너비 우선 탐색(BFS)

→ 큐

2) 깊이 우선 탐색(DFS)

→ 스택

3) 다익스트라(Dikstra)

→BFS응용 + 최소 힙

***효율적으로 해킹하기**

1) answer 배열 받기(최대값 비교)

2) 방문 배열 받기(방문→T / 방문X →F)

3) N만큼 반복→ ArrayList 초기화

4) M만큼 반복 → A 인접 리스트에 그래프 데이터 저장

5) N개수 만큼 반복 → BFS 탐색

6) BFS에서 answer배열  

7) answer 배열에서 가장 큰 수 찾기

8) N 개수 만큼 반복 → answer배열에서 max값을 가진 index 출력
