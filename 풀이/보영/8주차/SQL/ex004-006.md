
## 004 조건에 맞는 도서와 저자 리스트 출력하기
```sql
SELECT  b.BOOK_ID AS BOOK_ID,
        a.AUTHOR_NAME AS AUTHOR_NAME,
        TO_CHAR(b.PUBLISHED_DATE, 'yyyy-mm-dd') AS PUBLISHED_DATE
  FROM BOOK b
  JOIN AUTHOR a
   ON b.AUTHOR_ID = a.AUTHOR_ID
  WHERE b.CATEGORY = '경제'
  ORDER BY PUBLISHED_DATE ;
```

## 005 그룹별 조건에 맞는 식당 목록 출력하기
```sql
SELECT M.MEMBER_NAME, R.REVIEW_TEXT, DATE_FORMAT(R.REVIEW_DATE, "%Y-%m-%d") AS REVIEW_DATE
FROM MEMBER_PROFILE M INNER JOIN REST_REVIEW R
ON M.MEMBER_ID = R.MEMBER_ID
WHERE M.MEMBER_ID = (SELECT MEMBER_ID 
                     FROM REST_REVIEW
                 GROUP BY MEMBER_ID
                 ORDER BY COUNT(*) DESC LIMIT 1)
ORDER BY REVIEW_DATE, REVIEW_TEXT;
```

## 006 상품을 구매한 회원 비율 구하기 - 프로그래머스
```sql
SELECT YEAR, MONTH, COUNT(*) AS PUCHASED_USERS,
	ROUND((COUNT(*)/ (SELECT COUNT(*)
					FROM USER_INFO WHERE YEAR(JOINED) = 2021)), 1) AS PUCHASED_RATIO
FROM (
    SELECT DISTINCT YEAR(S.SALES_DATE) AS YEAR, MONTH(S.SALES_DATE) AS MONTH, U.USER_ID
    FROM ONLINE_SALE S
    JOIN USER_INFO U ON S.USER_ID = U.USER_ID AND YEAR(JOINED) = 2021
) A
GROUP BY YEAR, MONTH
ORDER BY YEAR, MONTH
```
